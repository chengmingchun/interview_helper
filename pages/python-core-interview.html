一段 Python 代码从写下到释放对象经历什么

1. 源码你写下变量、函数、装饰器、类。
2. 编译解析成 AST，再编译为字节码。
3. 执行帧函数调用创建 frame，局部变量是名字表。
4. 对象绑定名字绑定对象，赋值不是复制对象。
5. GIL 调度CPython 同时只有一个线程执行字节码。
6. 内存回收引用计数为主，GC 补循环引用。
